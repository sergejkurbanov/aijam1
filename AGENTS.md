# AGENTS.md — AI JAM #1

Single-page event landing for the AI JAM #1 meetup (9 Apr 2026, Prague).
**Stack:** React 19.2, TypeScript ~6.0, Vite 8, Tailwind CSS 4, React Compiler — SPA, no router.

## Commands

| Command | What it does | When to use |
|---------|-------------|-------------|
| `npm run dev` | Vite dev server at `localhost:5173/aijam1/` | Local development |
| `npm run build` | `tsc -b` + Vite production build → `dist/` | Before committing, CI |
| `npm run lint` | ESLint (TS + React Hooks rules) | Before committing |
| `npm run preview` | Serve production `dist/` locally | Verify build output |

## Architecture

**Data flow:** `index.html` → `src/main.tsx` (ReactDOM.createRoot) → `src/App.tsx` (renders full page)

- **No router** — single page, anchor links only (`#topics`, `#venue`, `#register`)
- **No state management** — static content, no interactivity beyond CSS hover
- **React Compiler** enabled (`babel-plugin-react-compiler`) — do NOT add `useMemo`/`useCallback`
- **Stitch design origin** — UI generated by Google Stitch MCP, converted from HTML to TSX manually
- `vite.config.ts` `base: '/aijam1/'` — required for GitHub Pages subfolder deployment; all asset paths must be relative
- Deployment: push to `main` → GitHub Actions builds → GitHub Pages publishes

## Conventions

- `verbatimModuleSyntax: true` → always use `import type` for type-only imports
- `noUnusedLocals` + `noUnusedParameters` enforced — clean up before building
- **CSS**: flat BEM-like class names (`.hero-title`, `.btn-primary`); design tokens as CSS custom properties in `src/index.css` `:root` block; no JS-side theme values
- **No Tailwind utility classes in JSX** — Tailwind is imported for reset/base only; all styling via named classes in `index.css`
- Data arrays defined as module-level `const` in `App.tsx`, not inside the component

```tsx
// Preferred pattern — data outside component, flat CSS classes
const items = [{ icon: 'bolt', title: 'AI Workflow' }]

function App() {
  return <div className="app-root">...</div>
}
```

## Design System (Stitch "Meow Modern")

CSS vars defined in `src/index.css` `:root`:

| Token | Value |
|-------|-------|
| `--color-bg` | `#f8f6f2` cream |
| `--color-primary` | `#00684f` mint green |
| `--color-secondary-container` | `#ffc2cc` pastel pink |
| `--ff-heading` | Plus Jakarta Sans |
| `--ff-body` | Be Vietnam Pro |

No 1px borders. Minimum border-radius `0.5rem`. Pill buttons (`border-radius: 9999px`).

## Boundaries

- ✅ Run `npm run lint && npm run build` before committing
- ✅ Keep all styles in `src/index.css`; keep all content in `src/App.tsx`
- ⚠️ Ask before adding npm dependencies — bundle size matters for a static landing
- ⚠️ Ask before changing `vite.config.ts` `base` — breaks all GitHub Pages asset paths
- ⚠️ Ask before splitting `App.tsx` into multiple components — current simplicity is intentional
- 🚫 Never commit API keys (Stitch key is in `.vscode/mcp.json` — verify it stays gitignored)
- 🚫 Never add `useMemo`/`useCallback` — React Compiler handles memoization automatically
